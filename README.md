MIDI-USB-SHIELD
===============

Build instructions
------------------
    git clone https://github.com/abcminiuser/lufa.git
    git clone https://github.com/openpipelabs/midi-usb-shield.git
    cd midi-usb-shield
    make

Firmware Update
---------------
Enter DFU mode powering the shield while HWB jumper is ON

    sudo make dfu
